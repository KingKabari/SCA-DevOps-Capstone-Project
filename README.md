# SCA-Terraform-Provisioning-Infrastructure

## Overview
This project demonstrates the use of Terraform to provision and manage AWS infrastructure.

---

## Files Included
- main.tf → Terraform configuration file  
- answers.txt → Answers to assignment questions  
- terraform-init.png → Screenshot of terraform init  
- terraform-plan-&-apply.png → Screenshot of terraform plan and apply  

---

## Steps Performed

### Initialize Terraform
```bash
terraform init
```
 
### Plan Infrastructure
```bash
terraform plan
```
### Apply Configuration
```bash
terraform apply -auto-approve
```
